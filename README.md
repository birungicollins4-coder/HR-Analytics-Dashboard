# HR Analytics Dashboard

## Project Overview

This HR analytics dashboard analyzes workforce data to uncover employee attrition trends, salary distribution, department performance, and employee demographics.

---

## Tools Used

- Power BI
- SQL
- Python
- Excel

---

## Key Insights

- High attrition observed in sales department
- Employees with lower satisfaction scores were more likely to leave
- Overtime strongly correlated with attrition
